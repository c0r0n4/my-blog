HTML Elements

An HTML element usually consists of a start tag and end tag, with the content inserted in between

The HTML <b>element</b> is everything from the start tag to the end tag:

HTML elements with no content are called empty elements. Empty elements do not have an end tag, such as the
element (which indicates a line break).

Nested HTML Elements

HTML elements can be nested (elements can contain elements).

All HTML documents consist of nested HTML elements.

This example contains four HTML elements:

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

The <b>&lt;html&gt;</b> element defines the <b>whole document</b>.

It has a <b>start</b> tag &lt;html&gt; and an <b>end</b> tag &lt;/html&gt;.

The element <b>content</b> is another HTML element (the &lt;body&gt; element).

<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

The <b>&lt;body&gt;</b> element defines the <b>document body</b>.

It has a <b>start</b> tag &lt;body&gt; and an <b>end</b> tag &lt;/body&gt;.

The element <b>content</b> is two other HTML elements (&lt;h1&gt; and &lt;p&gt;).

<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>

The <b>&lt;h1&gt;</b> element defines a <b>heading</b>.

It has a <b>start</b> tag &lt;h1&gt; and an <b>end</b> tag &lt;/h1&gt;.

The element <b>content</b> is: My First Heading.

<h1>My First Heading</h1>

The <b>&lt;p&gt;</b> element defines a <b>paragraph</b>.

It has a <b>start</b> tag &lt;p&gt; and an <b>end</b> tag &lt;/p&gt;.

The element <b>content</b> is: My first paragraph.

<p>My first paragraph.</p>
